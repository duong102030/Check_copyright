# NKD Copyright Pre-commit Hook

This pre-commit hook automatically adds a custom copyright header
to new source files (.c, .h, .cpp, .hpp, .ino).

## Example Header

```c
/*
 * example.c
 *
 *  Created on: November 1, 2025
 *      Author: Nguyen Kha Duong
 *      Contact via email: duong nguyen kha.daniel@gmail.com
 */
